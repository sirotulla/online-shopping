# Online Shop Project

## 📌 Description
Simple PHP-based online shop with:
- User authentication
- Wallet system
- Admin panel
- Order management

## ⚙️ Setup Instructions

1. Clone the repository
2. Import `database_schema.sql` into your MySQL database
3. Configure database connection in:
   app/config/database.php
4. Run project using Apache or Nginx
5. Set document root to the `public/` folder

## 🛠 Tech Stack
- PHP
- MySQL
- PDO
- HTML/CSS

## 🔐 Notes
Do not upload real database credentials.
Use fake values in database.php or environment variables.
